Why Frontloop

Stop switching tabs.
Start shipping faster.

The current workflow for UI changes with AI agents is broken. Frontloop fixes it end-to-end.

⚡

No more context-switching

Click the element you want to change — right in the browser. No more describing CSS selectors in text or hunting for class names in the IDE.

🎯

Token & time efficient

Full DOM context — selector, outer HTML, computed styles, component hierarchy — means Claude makes precise edits in one shot. No grep loops.

🔄

Real edits, not overrides

Claude edits the actual source file using full codebase context. Hot Module Replacement delivers the change instantly in your browser.

🧩

Framework aware

Component hierarchy is extracted automatically. React, Next.js, Angular — Claude knows exactly which component owns the element.

📸

Two selection modes

DOM element selector for precise single-element targeting, or screenshot selector to draw a box around any area and describe the change.

✅

Type-safe by default

Claude runs a type check after every edit. If types break, it fixes before hot-reloading. No silent regressions shipped.

How it works

Six steps from click to change

A floating button lives in your browser during development. Everything else is automatic.

STEP 01
🔘

FAB appears

A floating action button is injected in dev mode. Hover to reveal DOM or screenshot selector modes.

STEP 02
🖱️

Select element

Click an element to inspect it, or draw a box with screenshot selector to target any visual region.

STEP 03
✏️

Describe change

Type your desired change. An orange shimmer animation appears over the element while Claude works.

STEP 04
🔌

WebSocket streams

The full DOM payload is streamed to Claude Code in real-time via WebSocket — no polling.

STEP 05
🤖

Claude edits source

Claude receives full context, edits the source file directly, and runs a type check to verify correctness.

STEP 06
🚀

HMR delivers

The shimmer clears, HMR hot-reloads the change. If HMR won't catch it, a full reload signal is sent.

Full DOM Context

Claude sees exactly what you see

Every task payload includes the complete context Claude needs to make a precise, one-shot edit without back-and-forth.

  • ▸ Selector — the precise CSS path to the target element
  • ▸ Outer HTML — the actual markup of the element
  • ▸ Computed styles — live CSS values from the browser
  • ▸ Component hierarchy — React/Angular tree from element to root
  • ▸ Page context — current URL and page title

Get Started

Up in two commands

Install the Frontloop plugin for Claude Code. Run your dev server. Invoke the skill. That's it.

STEP 01 — ADD TO MARKETPLACE
/plugin marketplace add tamal-thetaonelab/frontloop

Registers the plugin with Claude Code's plugin system.

STEP 02 — INSTALL
/plugin install frontloop

Installs the plugin and makes skills available in your session.

STEP 03 — USE THE SKILL
/live-ui-generation

Start your dev server first, then invoke the skill. The FAB appears in your browser.

Requirements: Claude Code · Node.js 18+
